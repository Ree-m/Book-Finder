## Getting Started

This project was inspired by the repository APP IDEAS.To develop this web page, I used react and Google Books Api.

### Prerequisites

To use the software locally, you need
* Npm or Yarn
* React


### Installation
What you need to install to run the system locally

   ```sh
   npm install
   ```
   ```sh
   npm run start
   ```

   or if you use yarn

   ```sh
   yarn install
   ```
   ```sh
   yarn start
   ```
